\section{Introduction}
The code is reasonably organized: you can find the source files in src/, 
header files in inc/, object files and binaries in obj/ and bin/ 
respectively, and so on. Files related to this report are present in doc/.
Simulation results for the most recent run are in tests/ and archived
reults in archives/. 
Compiling is simple:
\begin{itemize}
	\item To compile the normal binary, do \\
		make [norm]  
		This creates bin/lo3mac, which relies on a pre-created recordFlow.txt.
		To run it, do: \\
		bin/lo3mac $\langle$error rate$\rangle$ <sim duration$\rangle$
	\item To compile the generator binary, do \\
		make gen
		This creates bin/gen\_lo3mac, which generates recordFlow.txt for future use.
		To run it do: \\
		bin/gen\_lo3mac $\langle$error rate$\rangle$ $\langle$sim duration$\rangle$ $\langle$call interval$\rangle$ $\langle$call duration$\rangle$ $\langle$stored voice duration$\rangle$
\end{itemize}
